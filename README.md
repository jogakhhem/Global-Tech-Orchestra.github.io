# GU Lost and Found Hub

## Project Overview
GU Lost and Found Hub is a web-based platform designed to help users report, search, and manage lost and found items efficiently. The website fosters community collaboration to reunite lost belongings with their owners through an easy-to-use interface.

## Features
- User registration and login for secure access
- Reporting lost and found items with detailed descriptions
- Search functionality to find reported items by category or keywords
- Dashboard for users to manage their reports
- Responsive design for accessibility on all devices
